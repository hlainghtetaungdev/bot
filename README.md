🤖 All-In-One Betting Control Panel Bot

ဒီ Project သည် SportsXZone, 555Mix, IBet789, Sports899 စတဲ့ Betting Site တွေရဲ့ Agent Panel တွေကို Telegram Bot ကနေတဆင့် အလိုအလျောက် ငွေသွင်း/ငွေထုတ် (Deposit/Withdraw) ပြုလုပ်နိုင်ရန် ဖန်တီးထားသော Automation Tool တစ်ခုဖြစ်ပါတယ်။

🛠️ Built With (အသုံးပြုထားသော နည်းပညာများ)

ဒီ Project ကို အောက်ပါ Language နှင့် Library များအသုံးပြုပြီး တည်ဆောက်ထားပါတယ် -

✨ Features (လုပ်ဆောင်နိုင်စွမ်းများ)

✅ Multi-Platform Support: Site ၄ ခု (SXZ, 555Mix, IBet, S899) ကို Bot တစ်ခုတည်းမှ ထိန်းချုပ်နိုင်ခြင်း။

✅ Auto Login & Navigation: Website များကို အလိုအလျောက် Login ဝင်ပြီး သက်ဆိုင်ရာ စာမျက်နှာသို့ သွားရောက်ခြင်း။

✅ Smart User Detection: Username ပေါ်မူတည်ပြီး မည်သည့် Site ဖြစ်သည်ကို အလိုအလျောက် ခွဲခြားသိရှိနိုင်ခြင်း (ဥပမာ - gg2 နဲ့စရင် Sports899, uu2 နဲ့စရင် 555Mix)။

✅ Secure Transaction: ငွေသွင်း/ငွေထုတ် လုပ်ငန်းစဉ်များကို တိကျမှန်ကန်စွာ ဆောင်ရွက်ပေးခြင်း။

✅ Receipt Generation: Transaction အောင်မြင်ပါက ငွေပြေစာ (Receipt) ကို ဓာတ်ပုံနှင့်တကွ ထုတ်ပေးခြင်း။

✅ Admin Security: ADMIN_ID ပါဝင်သော User များသာ အသုံးပြုခွင့်ရှိခြင်း။

🚀 Installation & Setup

Clone the Repo



Install Requirements
လိုအပ်သော Python Library များကို Install လုပ်ပါ။

pip install -r requirements.txt


(မှတ်ချက်: selenium, python-telegram-bot, python-dotenv, webdriver-manager တို့ လိုအပ်ပါမယ်)

Configuration (.env)
.env ဖိုင်တစ်ခု တည်ဆောက်ပြီး အောက်ပါအချက်အလက်များကို ဖြည့်သွင်းပါ။

BOT_TOKEN=your_telegram_bot_token
ADMIN_ID=your_telegram_id
# အခြားသော Site Username/Password များ...


Run the Bot

python bot.py
